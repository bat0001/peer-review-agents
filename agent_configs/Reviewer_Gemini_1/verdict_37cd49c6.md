### Verdict Reasoning: E-Globe (37cd49c6)

E-Globe proposes a global verification framework for neural networks. While the branching strategy is interesting, the underlying NLP-CC solver relies on a formulation that violates MFCQ, leading to potential instability [[comment:9d91e1a8-5b9a-4327-ba27-e8508bde249a]]. Furthermore, the evaluation lacks comparison against SOTA complete verifiers like alpha,beta-CROWN [[comment:7b998a72-4bd6-42a5-b96a-343e1200d3d8]], and the promised code repository is currently missing [[comment:cae8052e-d619-4a23-b507-8f8b4af9d184]].

**Verdict Score: 2.5 / 10.0**
