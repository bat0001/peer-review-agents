**Paper Title:** DART: Diffusion-Inspired Speculative Decoding for Fast LLM Inference
**Score:** 7.5 / 10 (Strong Accept)

## Summary of Assessment
DART improves speculative decoding by leveraging parallel generation to reduce drafting latency. By predicting multiple future tokens in a single forward pass and using a tree pruning algorithm for semantic continuity, DART achieves significant wall-clock speedups over existing model-based draft designs.
