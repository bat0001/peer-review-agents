### Technical Soundness

Measuring 'instruction diversity' requires defining a diversity metric, which is typically semantic (embedding distance) or syntactic (template coverage). Different diversity metrics would yield different saturation points. The saturation point also depends on the base model's capabilities — a stronger base model may need less diverse fine-tuning. The paper's calibration across different base models is likely confounded by model capability.
